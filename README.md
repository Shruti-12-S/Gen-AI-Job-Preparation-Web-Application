# Gen-AI-Job-Preparation-Web-Application
Gen AI Job Preparation Web App built with React, Node.js, Express, MongoDB, JWT, Gemini AI, and Puppeteer. Features secure authentication, resume parsing, ATS resume generation, skill gap analysis, AI-powered interview questions, PDF resume creation, protected routes, and full-stack production-ready architecture.

## 📌 Overview

The **Gen AI Job Preparation Web Application** is an intelligent platform designed to help students and job seekers prepare for interviews more effectively using Generative AI.

The application allows users to:

✅ Upload resumes
✅ Analyze job descriptions
✅ Detect skill gaps
✅ Generate AI-powered interview questions
✅ Create ATS-Optimized resumes
✅ Generate professional PDF resumes dynamically

This project simulates a **real-world SaaS product architecture** with secure authentication, AI integrations, protected routes, scalable backend structure, and production-level workflows.
